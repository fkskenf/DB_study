# Oracle

> - START WITH / CONNECT BY
>> 1. START WITH 절에 시작 조건을 찾습니다.
>> 2. CONNECT BY 절에 연결조건을 찾습니다.
```java
SELECT [컬럼]...
FROM [테이블]
WHERE [조건]
START WITH [최상위 조건]
CONNECT BY [NOCYCLE][PRIOR 계층형 구조 조건];
```
```java
SELECT 
DEPT_NAME,
DEP_CD,
PARENT_CD,
LEVEL
FROM DEP 
START WITH PARENT_CD IS NULL --최상위노드 설정,
CONNECT BY PRIOR DEP_CD = PARENT_CD;--부모노드와 자식노드 연결
```
